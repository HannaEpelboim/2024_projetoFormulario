Projeto REGEX

Sobre o projeto

O projeto envolveu o desenvolvimento desse site de formulários, com foco na aplicação prática do aprendizado em HTML, CSS e JavaScript, especialmente utilizando expressões regulares (regex). O principal objetivo do site era garantir que as informações inseridas fossem validadas corretamente de acordo com as regras estabelecidas.
Para a construção do site, foram utilizados HTML e CSS para estruturar e estilizar a interface, e JavaScript para as validações.
O projeto foi realizado por Judy e Hanna, com o auxílio das tutoras Jéssica e Érica, além do apoio do monitor Andrey e Maria.
